GeoNode is an open source platform for sharing geospatial data and maps.

You can get more information here.


Three demo instances are available and open for registration:

Latest stable

4.2.x

Development

master

You can register if you want to upload your own data or if you want to edit public data; data will be wiped out once a week as we restore the demo to a clean state, but users' accounts will be maintained across such weekly restores.


If you encounter any issues, please, report them using the users' mailing list (see next paragraph) as you will help us improving the demo and GeoNode itself.


If you have questions or additional feedback, join our users' mailing list.
